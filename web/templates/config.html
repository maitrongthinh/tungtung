📋 Config Guide

integrations.web_secret_key: Mat khau dang nhap dashboard. Dung python -c "import secrets; print(secrets.token_urlsafe(32))" de tao.

integrations.ai_api_key: API key cho AI (OpenAI/Gemini/custom). Hoac config ai_providers de dung nhieu key.

integrations.shopee_affiliate_cookie: Cookie Shopee de tao affiliate link. Copy tu browser sau khi dang nhap Shopee Affiliate.

meta_app_id + meta_app_secret: Facebook App credentials de refresh token tu dong.

ai.max_daily_requests: Gioi han so luong API call AI moi ngay. Mac dinh 200.

kpi.posts_per_day: So bai dang muc tieu moi ngay. Mac dinh 20.

Luu y: storage section khong hien thi tren web de tranh thay doi duong dan DB khi he thong dang chay.

🍪 Shopee Affiliate Cookie

Paste JSON cookie từ extension (Cookie Editor / EditThisCookie). Cookie hết hạn sau vài tuần — update tại đây không cần restart.

🌐 Proxy List

Mỗi dòng một proxy. Format: http://user:pass@host:port hoặc http://host:port. Bot xoay vòng tự động khi crawl Shopee.

🔗 Kiểm tra URL Shopee

Paste link sản phẩm Shopee để kiểm tra có đúng định dạng không và test generate affiliate link.

🤖 AI Providers

Thêm nhiều key/provider — bot tự xoay vòng khi bị rate limit hoặc quota hết.

▶ Hướng dẫn lấy key
Google AI Studio (Gemini miễn phí):
1. Vào aistudio.google.com/apikey → Create API key
2. Chọn provider = gemini, dán key vào
3. Model gợi ý: gemini-2.0-flash (nhanh, miễn phí 1500 RPD)

TrollLLM (Claude miễn phí):
Provider = trollllm, Base URL = https://chat.trollllm.xyz/v1
Model: claude-sonnet-4-6 — giới hạn ~1500 RPD, 16 RPM

Chiến lược xoay vòng: Thêm 2-3 key Gemini + 1 key TrollLLM → khi Gemini hết RPM (16/phút), bot tự chuyển sang TrollLLM và ngược lại. Không bị gián đoạn.

📘 Facebook Token

Dán User Access Token để đổi lấy Page Access Token và cập nhật vào account.
